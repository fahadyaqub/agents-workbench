# Agents Workbench

This directory contains the shared system behind Agents Workbench.

## Structure

- `shared/` contains team-wide guidance, domains, workflows, and memory
- `local/` contains personal, machine-local overlays
- `templates/` contains starter files used during setup and project bootstrap

## Principles

- `AGENTS.md` is the single source of truth for agent instructions wherever possible
- Project `AGENTS.md` files are shared and committed when appropriate
- Shared guidance should never erase project-specific guidance
- Existing `AGENTS.md`, `CLAUDE.md`, `CODEX.md`, and workflow docs must be preserved and merged carefully
- Shared memory should contain durable, reusable knowledge rather than task noise
